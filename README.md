# Custom Alert Buttons

This project is a React Native application built with Expo that displays a toolbar containing three custom alert buttons. Each button shows a unique alert message when pressed, demonstrating how to pass props and handle basic interactivity in a component-based app.

## Features
- Custom AlertButton component with dynamic alert messages

- Toolbar component rendering multiple alert buttons

- Demonstrates prop usage and component composition

## Installation
- Clone the repo:
  - git clone `https://github.com/katrina-l-hill/custom_alert_buttons`
  - cd alert-buttons
  - npx expo start
